Skills (🔥)

HTML

HyperText Markup Language

É uma linguagem de marcação utilizada na construção de páginas na Web. Define o significado e a estrutura do conteúdo. A linguagem é responsável pelas adições de título, paráfrafos, imagens e muitos outros conteúdos.

Css

Cascading Style Sheets

O CSS é quem dá a estilização da página web, definindo cores, fontes, tamanhos, posicionamento e qualquer outro valor estético para os elementos da página.

JavaScript

Linguagem de Programação

O JavaScript é uma linguagem de programação interpretada estruturada, e tem como principal objetivo criar páginas dinâmincas, ele dá a interatividade que a página precisa, é uma das linguagens mais utilizadas no mundo.

Ruby

Linguagem de Programação

Ruby é uma linguagem de programação interpretada multiparadigma, de tipagem dinâmica e forte, com gerenciamento de memória automático, originalmente planejada e desenvolvida no Japão em 1995, por Yukihiro "Matz" Matsumoto, para ser usada como linguagem de script.

Git

Versionador de código

Git é o sistema de controle de versão open source mais usado no mundo atualmente! Ele é usado para controlar o histórico de alterações de arquivos e principalmente de projetos de desenvolvimento de software. Ele permite mais flexibilidade no fluxo de trabalho, segurança e desempenho.

Sql

Linguagem de Programação

SQL significa Standard Query Language, literalmente a linguagem padrão para realizar queries. A linguagem SQL é utilizada de maneira relativamente parecida entre os principais bancos de dados relacionais do mercado: Oracle, MySQL, MariaDB, PostgreSQL, Microsoft SQL Server, entre muitos outros.

Figma (Basic)

Software de Protótipos

Figma é um editor gráfico de vetor e prototipagem de projetos de design baseado principalmente no navegador web, com ferramentas offline adicionais para aplicações desktop para GNU/Linux, macOS e Windows.

Node Js

FrameWork

O Node. js é uma tecnologia usada para executar o JavaScript server-side. Isso quer dizer que ele permite a execução da linguagem JavaScript fora do navegador. Nesse sentido, com essa tecnologia, é possível construir diversas aplicações Web, como websites, APIs e microsserviços.

React

FrameWork

React é um framework JavaScript criado pelo Facebook (atual Meta) que é usado para criar interfaces de usuário (UI) em aplicativos web. Ele é popular por ser fácil de usar, altamente flexível e escalável, e é usado por muitas empresas de tecnologia, incluindo o Facebook, Instagram e Airbnb.

Bootstrapt

FrameWork

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.
